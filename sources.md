# 《地铁修建看城市温度》数据来源说明

更新时间：2026-06-12

## 官方/权威来源

1. 国家发展改革委：《国家发展改革委关于山东省济南市城市轨道交通第二期建设规划（2020-2025年）的批复》
   - 使用数据：二期规划 6 条线路、约 159.6 公里、估算总投资 1154.36 亿元；批复中关于泉水保护、文物保护、枢纽衔接的要求。
   - 链接索引：https://www.ndrc.gov.cn/

2. 济南市统计局：《2025年济南市国民经济和社会发展统计公报》
   - 使用数据：地区生产总值 14210.09 亿元，常住人口 961.6 万人，常住人口城镇化率 77.3%，规模以上高技术制造业增加值增长 14.7%；4号线、8号线、6号线东段、济阳线、高新东区云巴通车运营。
   - 链接：https://jntj.jinan.gov.cn/col/col18254/art/2026/art_7dc3135bb2209f961b3c65baa8ab3d2d.html

3. 济南轨道交通集团公开资料
   - 使用数据：线路开通、运营、线路长度与线网叙事。
   - 链接：http://www.jngdjt.cn/

4. 济南市人民政府公开信息
   - 使用数据：轨道交通建设、开通运营、城市交通和民生叙事。
   - 链接：https://www.jinan.gov.cn/

5. 济南市城乡水务局水位专题页
   - 使用数据：趵突泉、黑虎泉等泉水水位发布口径；趵突泉地下水位预警线。
   - 链接：https://jnwater.jinan.gov.cn/
   - 说明：水位为动态数据，本作品只呈现公开预警阈值与历史公开报道节点，实时值以主管部门每日发布为准。

6. 中国文明网及全国文明城市公开名单
   - 使用数据：济南“全国文明城市”指标说明。
   - 链接：http://www.wenming.cn/

## 辅助报道来源

1. 新华社：《考古新发现将济南建城史提前约1500年》
   - 使用内容：大明湖站附近发现自大汶口文化时期至近现代遗址和文物；大明湖站因文物保护调整设计与施工工法。
   - 链接：https://www.news.cn/local/20251203/ef5be7073e634edc88777d993833c394/c.html

2. 央广网、人民日报客户端等关于趵突泉水位和连续喷涌的公开报道
   - 使用内容：趵突泉水位预警线、阶段性高水位和连续喷涌背景。
   - 说明：用于补充解释，不替代水务主管部门实时水位发布。

## 数据口径说明

- `data/jinan-metro-data.json` 为评审和复核用数据文件。
- `data/jinan-metro-data.js` 为网页本地打开时使用的数据副本，内容由 JSON 文件机械生成。
- 轨道交通运营里程涉及多条线路在 2025 年末集中通车，页面采用公开线路长度累加和统计公报信息整理，文字中避免写成未经官方统一发布的精确“官方总里程”。
- “红山文化”未被采用。可核验权威资料显示，大明湖站相关考古发现表述为“大汶口文化时期至近现代遗址和文物”，页面已据此修正。
