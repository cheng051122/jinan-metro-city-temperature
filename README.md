# 地铁修建看城市温度

这是《地铁修建看城市温度》数据新闻网页的 GitHub Pages 发布版本。

## 文件说明

- `index.html`：完整单文件网页，已内嵌封面图和数据，可直接作为静态页面发布。
- `sources.md`：数据来源说明。
- `.nojekyll`：让 GitHub Pages 按静态文件原样发布。

## 发布方式

把本文件夹内的文件上传到一个公开 GitHub 仓库根目录，然后在仓库 `Settings -> Pages` 中选择 `Deploy from a branch`，分支选 `main`，目录选 `/root`。

发布成功后，网址通常是：

`https://你的GitHub用户名.github.io/仓库名/`
